Encoding



Encoding is the process of taking in information from the world around us and transforming it into a form our brain can use and store.


Atkinson-Shiffrin Model

  • Sensory Memory: Brief retention of sensory details like a fleeting image.
  • Short-Term Memory: Holds information temporarily, typically about 7 items for 20 seconds.
  • Long-Term Memory: Information that gets stored indefinitely with enough rehearsal.

Automatic vs. Effortful Processing

Automatic Processing: Happens without realizing, such as remembering where you placed your phone.

Effortful Processing: Requires focus and attention, like studying for a test.


Explicit vs. Implicit Memories

Explicit Memories: Memories of facts or events you can consciously recall.

Implicit Memories: Unconscious skills or conditioned responses, like riding a bike.


Effortful Processing Strategies

Spacing Effect: Studying over time is better for retention than cramming.

Shallow vs. Deep Processing: Shallow focuses on surface details, while deep connects information to meaning.

Storage



Storage refers to how our brain holds onto information over time, either for a short while or indefinitely.


Sensory Memory

Iconic Memory: Visual info lasting a fraction of a second.

Echoic Memory: Auditory info lasting 3-4 seconds.


Short-Term/Working Memory

Short-term memory holds information for a brief time, while working memory allows us to manipulate that information (like solving problems in our heads).


Long-Term Memory

Long-term memory stores information for long periods of time.

  • Explicit Memory System (Hippocampus): Responsible for processing facts and experiences.
  • Implicit Memory System (Cerebellum): Stores procedural skills like riding a bike.

The Amygdala & Flashbulb Memories

Flashbulb Memories: Vivid, detailed memories of emotionally significant events.


Interference

Proactive Interference: Old information interferes with learning new information.

Retroactive Interference: New information makes it harder to remember old information.

Retrieval



Retrieval is the process of accessing stored memories.


Measuring Retention

  • Recall: Pulling information from memory without cues.
  • Recognition: Identifying information when presented with it.
  • Relearning: Learning information faster the second time around.

Misinformation Effect

This occurs when misleading information distorts your memory of an event.


Source Amnesia

Remembering information but forgetting where or how you learned it.


Retrieval Cues

Priming: Activating certain associations in memory without conscious awareness.

Mood/State Congruent Memory: Easier to remember things when in the same mood as when learned.

Context-Dependent Memory: Memory improves when in the same context as when you learned the information.


Serial Position Effect

The tendency to remember items at the beginning and end of a list better than the middle.
